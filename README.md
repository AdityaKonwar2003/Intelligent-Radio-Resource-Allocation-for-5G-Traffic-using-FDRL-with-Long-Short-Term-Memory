# Intelligent-Radio-Resource-Allocation-for-5G-Traffic-using-FDRL-with-Long-Short-Term-Memory
This is my major project of my B.Tech ECE  final year for 5G Traffic analyzation, validation and resource allocation.

# Overview

This project presents an AI-driven framework for intelligent radio resource allocation in 5G network slicing. The proposed system integrates Long Short-Term Memory (LSTM) for traffic prediction, Graph Neural Networks (GNN) for spatial relationship modeling, and Double Deep Q-Network (DDQN) reinforcement learning for adaptive resource allocation across eMBB, URLLC, and mMTC network slices.

The framework aims to improve Quality of Service (QoS) by learning resource allocation strategies that balance throughput, latency, fairness, and resource utilization under dynamic traffic conditions.
